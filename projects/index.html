Software development is not only my day job but also one of my biggest passions. Some of the software projects I pour time into outside of work are:

simskij/awesome-software-architecture

A curated list of resources on software architecture and related practices. To quote the repository readme: > Software architecture aims to describe the high level structures of software as well as the discipline of > creating them. As this topic is pretty broad, it might contain resources also present in more niche lists.

Star

containrrr/shoutrrr

Shoutrrr is a notification package for go applications, heavily inspired by the python library caronc/apprise. It is also usable as a standalone CLI tool. Maintained by the Containrrr team.

Star

containrrr/watchtower

Watchtower is currently the most popular solution for automating the updates of base images for docker containers. The project is written exclusively in Go. Maintained by the Containrrr team.
